# 📏 图片测量尺 - 爱硅蜜
**中文版本**
由爱硅蜜（AI Guimi）开发的一款轻量浏览器端图片测量工具。
支持通过快递单、身份证、银行卡等参照物，快速测量图片中的长度、角度，并将像素值换算为真实厘米/毫米。

---

## ✨ 核心功能
- 📐 **像素驱动测量**：基于图片坐标，直接计算两点距离和三点角度。
- 📏 **参照物换算**：输入已知物体的实际长度，一键校准像素/厘米比例。
- 🚀 **免安装即用**：纯前端单文件，浏览器直接打开，无需下载。
- ♻️ **开源轻量**：无依赖、无冗余代码，完全可自定义修改。

## 📖 使用说明
1. 打开工具链接：`https://aiguimi.github.io/image-measure-tool/`
2. 上传包含参照物的图片（例如长度为10cm的快递单）。
3. 点击参照物的两端，标记起点和终点。
4. 输入参照物的实际长度（厘米/毫米），完成比例校准。
5. 点击图片中的其他位置，即可测量任意物体的真实长度。
# 📏 Image Measure Tool - AIGuimi

---

**English Version**
A simple, browser-based image measuring tool powered by AI Guimi.  
Measure lengths, angles, and convert pixels to real-world units (cm/mm) using reference objects like express packages, ID cards, or bank cards.

---

## ✨ Features
- 📐 **Pixel-Based Measurement**: Calculate lengths and angles using coordinates from any uploaded image.
- 📏 **Unit Conversion**: Set a reference length to convert pixels to centimeters or millimeters.
- 🚀 **No Installation Needed**: Runs directly in your browser, no downloads required.
- ♻️ **100% Open-Source & Lightweight**: Clean, single-file HTML code.

## 📖 How to Use
1. Open the [live demo](https://aiguimi.github.io/image-measure-tool/) in your browser.
2. Upload an image containing a known reference object (e.g., a 10cm-long express label).
3. Click two points on the reference object to mark its ends.
4. Enter the actual length of the reference (in cm/mm) to calibrate the scale.
5. Click two more points to measure the real length of any object in the image.
