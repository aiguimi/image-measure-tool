# 📏 图片测量尺 - 爱硅蜜（AIGuimi）
一款轻量浏览器端图片测量工具，由 **爱硅蜜（AI Guimi）** 开发，支持通过快递单、身份证、银行卡等参照物，一键将像素值换算为真实厘米/毫米，无需下载，打开即用！

---
## ✨ 核心功能
- 📏 **长度测量**：基于图片坐标，直接计算两点间距离，支持像素/厘米/毫米单位切换
- 📐 **角度测量**：三点定位法，自动计算夹角度数，精度拉满
- 🟩 **面积测量**：多边形绘制模式，一键计算闭合图形的实际面积（平方厘米/平方毫米）
- 🎯 **参照物校准**：输入已知物体的实际长度，一键校准像素/厘米比例，测量结果精准可控
- 🌐 **无安装即用**：纯前端单文件，浏览器直接打开，无需下载、无需联网
- 🔧 **完全开源**：代码可自定义修改，适配你的专属使用场景

---
## 📖 使用说明
1.  上传图片：点击上传按钮，导入需要测量的图片
2.  参照物校准（可选但推荐）：框选已知尺寸的物体，输入实际长度，完成比例校准
3.  选择测量模式：
    - 长度：点击两点，自动显示距离
    - 角度：点击三点，自动计算夹角
    - 面积：连续点击绘制多边形，闭合后自动计算面积
4.  导出/记录结果：直接复制数值，或截图保存测量结果

# 📏 Image Measure Tool - AIGuimi
A lightweight browser-based image measurement tool developed by **AIGuimi**. Convert pixels to real-world units (centimeters/millimeters) using reference objects like express waybills, ID cards, or bank cards. No downloads required, works directly in your browser!

---
## ✨ Core Features
- 📏 **Length Measurement**: Calculate the distance between two points with pixel/cm/mm unit support
- 📐 **Angle Measurement**: 3-point positioning for accurate angle calculation
- 🟩 **Area Measurement**: Draw polygons to calculate the real area of closed shapes (cm²/mm²)
- 🎯 **Reference Calibration**: Input known object dimensions to calibrate pixel-to-centimeter ratio
- 🌐 **No Installation Needed**: Pure frontend single file, runs directly in any browser
- 🔧 **Open Source**: Fully customizable code for your specific use cases

---
## 📖 Usage Guide
1.  Upload Image: Click to upload the image you need to measure
2.  Calibrate with Reference (Optional but Recommended): Select a known-size object and input its actual length to calibrate the scale
3.  Select Measurement Mode:
    - Length: Click two points to display the distance
    - Angle: Click three points to calculate the included angle
    - Area: Draw a polygon with multiple clicks, the area is calculated automatically when closed
4.  Export Results: Copy values directly or save screenshots of measurements
